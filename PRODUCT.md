# Product

## Register

product

## Users

Hackathon judges, founders, investors, analysts, and engineering leads running "Intelligence Missions": they state a goal in plain language and watch an autonomous agent swarm sense the open web, weave a verified memory graph, and propose next actions. Context: live demos and focused desk research sessions, mostly on large screens, often in dim rooms (Microsoft Build AI Hackathon, theme: Agentic Web).

## Product Purpose

NebulaX is a living team-intelligence fabric. It turns a one-sentence mission into a self-correcting evidence graph with traceable claims, contradiction detection, human-veto corrections that propagate confidence, future-scenario forecasting, and executable proposed actions. Success: a viewer instantly understands what the swarm found, trusts the provenance, and can act in one click.

## Brand Personality

Precision-instrument, alive, confident. The interface should feel like a premium intelligence console (Linear's restraint x Bloomberg-terminal density x a mission-ops room): dark, luminous, calm under load, with motion that conveys state rather than decoration.

## Anti-references

- Generic SaaS dashboard gloss: cream backgrounds, identical card grids, gradient-text heroes.
- "Hacker movie" kitsch: green-on-black matrix rain, fake terminal noise, glitch effects.
- Over-decorated glassmorphism on every surface; blur is reserved for overlays and the workspace chrome.
- Cramped, scroll-broken panels where content is cut off (the current failure mode being fixed).

## Design Principles

1. **Evidence first** - the fabric (graph, radar, timeline) is the hero; chrome recedes, data glows.
2. **Motion = state** - every animation reports something real (agent working, confidence shifting, new signal). 150-250ms, ease-out, reduced-motion respected.
3. **One glance, one truth** - confidence color (emerald/amber/rose) means the same thing in every view; the inspector is the single place to read detail and act.
4. **Earned density** - Bloomberg-grade information density with Linear-grade alignment: tabular numerals, mono for telemetry, strict spacing rhythm.
5. **Demo-proof** - every view has an educational empty state, 60fps interactions, and works fullscreen.

## Accessibility & Inclusion

Target WCAG AA contrast for body text in both themes. Confidence is never encoded by color alone (always paired with a % figure). All animations have `prefers-reduced-motion` fallbacks. Interactive targets >= 28px, keyboard focus visible.
